CRUD Operations

  • List Posts: View all 100 posts in a table with User ID, Title, and Body
  • Create Post: Form with validation (User ID required, Title min 5 chars, Body min 10 chars)
  • Edit Post: Minimizable modal popup that persists across navigation
  • Show Post: Detailed view with all post information and edit button
  • Delete Post: Remove posts (changes won't persist - demo API)

Modal Minimize/Maximize

The edit modal includes an innovative minimize feature:

  • Click minus (-) to minimize â†’ Modal becomes a small blue widget at bottom-right
  • Background becomes fully interactive - you can navigate, view posts, etc.
  • Modal persists across pages - stays minimized even when you navigate
  • Click maximize (â›¶) to restore â†’ Full modal with your edits preserved

How Background Access Works

When minimized, the component switches from <Modal> to <Box>:

// When minimized - renders as Box (non-blocking)
if (isMinimized) {
  return <Box>...</Box>;
}

// When maximized - renders as Modal (blocking)
return <Modal>...</Modal>;
