üîç Technical Highlights

Modal Minimize Feature:

The edit modal uses React Context to persist across navigation. When minimized, it renders as a fixed Box component (non-blocking) instead of a Modal component (blocking).

Solution to Background Blocking:

By conditionally rendering <Box> when minimized instead of <Modal>, we bypass the Modal's built-in focus trap and event blocking, allowing full interaction with the background.

if (isMinimized) {
  return <Box>...</Box>; // Non-blocking!
}
return <Modal>...</Modal>; // Blocks normally
