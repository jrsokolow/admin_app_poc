üåü Unique Features

Minimizable Edit Modal

How it works:

  • When maximized: Uses Chakra UI Modal (blocks background)
  • When minimized: Renders as Box component (non-blocking)
  • Persists across navigation using React Context
// Conditional rendering based on state
if (isMinimized) {
  return <Box>...</Box>; // Non-blocking!
}
return <Modal>...</Modal>; // Blocks normally

  • Form with User ID, Title, and Body fields
  • Validation: Title min 5 chars, Body min 10 chars
  • Success notification on creation

  • Opens as modal popup (no page navigation)
  • Click minimize (-) to minimize to corner
  • Background remains fully interactive when minimized
  • Modal persists when navigating inside the app
  • Click maximize (‚õ∂) to restore
